Visibility Tile Flip

This project was our response to the project brief, which was to work on a prototype for improving accessibility or inclusiveness in cities. As a group, we decided to raise awareness for the visually impaired.

The concept consists of a large display that can be interacted with via mobile device by scanning a nearby QR code.

My main responsibility for the project was to code the prototype.
